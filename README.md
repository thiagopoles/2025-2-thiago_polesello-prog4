## Projeto colaborativo em PHP com integração de banco de dados (MySQL).
Esse trabalho foi feito em conjunto com os colegas Heitor e Isabely, sob orientação do professor Alexandre. Ele tem como foco demonstrar como aplicações web podem se comunicar com um banco de dados para persistir informações, realizar consultas e manter estados — algo essencial em desenvolvimento back-end.
É uma excelente vitrine de aplicação prática de PHP, SQL e lógica de negócios, além de uma oportunidade de mostrar trabalho em equipe e integração entre partes da aplicação.
